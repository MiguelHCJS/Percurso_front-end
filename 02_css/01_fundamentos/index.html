Regras CSS

CSS é a abreviação do termo em inglês Cascading style sheet, traduzindo seria folha de estilo em cascata. É um mecanismo que aplica estilo aos documentos web, como modificar fontes, cores, tamanhos e etc...


            Regra CSS

            seletor { propriedade: valor; }

            ߛ regra 
            ߛ seletor: é o alvo da regra CSS
            ߛ declaração: parâmetros de estilização (propriedade e valor)
            ߛ propriedade: define a caracteristica do seletor que será estilizada
            ߛ valor: é a quantificação ou qualificação da propriedade

A regra pode ser usada toda em uma linha:


            seletor { color: red; font-size: 16px; width: 200px;}

Ou cada uma em sua linha:


            seletor {
                color: red;
                font-size: 16px;
                width: 200px;
            }

É apenas uma questão de torná-la mais legível e organizada, usar cada uma em sua linha. Quando o valor de uma propriedade for uma palavra composta, separada por espaço, usa-se (" ") ou (' '):


            seletor {
                font-family: "courier new";
            }

            seletor {
                font-family: 'courier new';
            }

Espaços multiplos serão tratados como espaços simples:


            seletor { border:   1px  solid      red;}
            
            Assim sendo

            seletor { border: 1px solid red;}

Assim como não é sensível ao tamanho da caixa das fontes(maiúsculas e minúsculas), resultará no mesmo efeito:


            SELETOR { BORDER: 1PX SOLID RED;}

Os comentários dentro do arquivo CSS é utilizado as marcações:
- inicio do comentário: /*
- fim do comentário: */
- /* comentário */

UNIDADES DE MEDIDA

COMPRIMENTO

Diz respeito às medidas vertical e horizontal, na qual pode ser declarada de diversas formas, e se em caso de valor 0, não há a necessidade de especificar o tipo de medida usada.

MEDIDA RELATIVA

São as medidas que tomam seu valor com base em outra medida de referência, sendo possível sua escalonagem para dispositivos diferentes:

Alguns exemplos:

UNIDADE DEFINIÇÃO
em relativa a font-size padrão do elemento, ou elemento-pai.
px referente ao dispositivo de renderização(onde está sendo visualizado o elemento)
vw referente à largura do viewport
vh referente à altura do viewport

MEDIDA ABSOLUTA

São medidas que os valores especificados são fixos, útil quando se está focada em uma medida de dispositivo especifico.

Alguns exemplos:

UNIDADE DEFINIÇÃO
mm milímetro
cm centímetro
in polegada, 1 polegada = 2,54 cm
px pixel, 1px = 1/96polegada

PORCENTAGEM

É uma medida que toma outra de referencia do elemento ancestral, elemento pai, declarado com valor seguido de %: width: 50%;

Outras unidades de medida(ângulo, hora, frequência e resolução) e o complemento das anteriores, no site do MAUJOR.

VINCULANDO FOLHA DE ESTILO NO DOCUMENTO

Três formas de se fazer este vinculo ao estilo, são eles:

Estilo inline

Este modo, é colocado as declarações direto em uma tag HTML:


            <p style='width: 100px; color: red;'>inline</p>

Este tipo de declaração dificulta a manutenção, retirando também a capacidade do CSS de ter um controle centralizado das mudanças.

Estilos incorporados

É declarado o estilo em uma tag 'style' que fica no próprio documento na tag 'head':


        <head>
            <meta charset='UTF-8'>
            <title>Estilo incorporado</title>
            <style>
                width: 100px;
                color: red;
            </style>
        <head>

Nessa declaração, melhorasse a localização da estilização, porém ainda não resolve a questão do controle centralizado.

Estilos externos

Para vincular uma folha de estilo externa é preciso por o elemento 'link' na parte do 'head', da seguinte maneira:


            <link rel='stylesheet' href='local do arquivo CSS'>

E em uma folha de estilo externa, a declaração será mais objetiva para as regras CSS, usando os diversos tipos de seletores.
