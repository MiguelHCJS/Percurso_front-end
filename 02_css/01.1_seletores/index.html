SELETORES CSS

São as formas que podemos selecionar(encontrar) os elementos que vamos estilizar, na qual podemos dividir em algumas categorias:

  • Seletores simples(seleciona pelo nome, id ou classe)
  • Seletores combinados(seleciona com base na relação entre os elementos)
  • Seletores de pseudoclasse(Seleciona por um determinado estado)
  • Seletores de pseudoelementos(Seleciona e estiliza parte de um elemento)
  • Seletores de atributo(Seleciona com base no atributo ou seu valor)
SELETORES SIMPLES

Selecionar um elemento com base em seu nome, por exemplo:


            Um elemento de "parágrafo", marcamos em HTML com a tag 'p', sendo assim selecionado na folha CSS:

            p {
                regras CSS
            }

Selecionar usando a classe, em uma tag colocamos um atributo chamado classe, e assim chamamos em CSS, usando ".":


            html
            <p class='seletorClass'></p>

            css
            .seletorClass {
                regras CSS
            }

Sendo também possível selecionar um elemento específico com classe específica:


            p.seletorClass {
                regras CSS
            }

Os elementos podem receber mais de uma 'class':


            html
            <p class='nome1 nome2'></p>
            
            Obs: O nome da classe não pode começar com número.

Selecionar usando a id, em uma tag colocamos um atributo chamado id, e assim chamamos em CSS, usando "#":


            html
            <p id='seletorId'></p>

            css
            #seletorId {
                regras CSS
            }

            Obs: Assim como o 'class' o 'id' também não pode começar com número.

Existe também o seletor universal, que seleciona todos os elementos na página HTML, usando "*":


            * {
                regras CSS
            }

SELETORES COMBINADOS

É o tipo de seletor que determina uma relação entre os seletores:

  • Seletor descendente (espaço)
  • Seletor filho (>)
  • Seletor irmão adjacente (+)
  • Seletor irmão geral (~)

Seletor descendente referesse a todos os elementos dentro do outro:


            Ex: 'p' descendente de uma 'section':

            html
            <section>
                <p></p> 
                <div>
                    <p></p>
                </div>
                <p></p>
            </section>

            E para selecionarmos os elementos 'p' em um 'section':

            css
            section p {

            }

Seletor filho referesse a elementos filhos diretos de um elemento:


            html
            <section>
                <p></p> 
                <div>
                    <p></p>
                </div>
                <p></p>
            </section>

            css
            section > p {

            }

Seletor irmão adjacente referesse ao primeiro elemento correspondente:


            html
            <section>
                <div>
                    <p></p>
                </div>
                <p></p>
                <p></p>
                <p></p>
                <div>
                    <p></p>
                </div>
                <p></p>
                <p></p>
                <p></p>
            </section>

            css
            div + p {

            }

Seletor irmão geral referesse aos elementos seguinte a outro elemento:


            html
            <section>
                <p></p>
                <p></p>
                <div>
                    <p></p>
                </div>
                <p></p>
                <div>
                    <p></p>
                </div>
                <p></p>
                <p></p>
                <p></p>
            </section>

            css
            div ~ p {

            }

SELETORES DE PSEUDOCLASSE

Usado para definir um estado especial em um elemento, como por exemplo ao passar o mouse sobre o elemento, quando for clicado, quando estiver sendo o foco, etc... Não demonstrarei todos, mas alguns básicos:

Usando o ':link', altera a estilização do link(neste caso 'a'):


            html
            <a href='#' class='teste'>TESTE</a>

            css
            .teste:link {
                color: blue;
                font-size: 20px;
                font-weight: bolder;
                text-decoration: none;
            }

            RESULTADO

            TESTE

Usando o ':visited', ao clicar no link, tornando-o visitado:


            html
            <a href='#' class='teste02'>TESTE</a>

            css
            .teste:visited {
                color: gray;
            }

            RESULTADO

            TESTE

Usando o ':hover', passando o mouse sobre o elemento:


            html
            <a href='#' class='teste'>TESTE</a>

            css
            .teste:hover {
                color: yellowgreen;
            }

            RESULTADO

            TESTE

Usando o 'active', ao clicarmos e manter pressionado o elemento:


            html
            <a href='#' class='teste'>TESTE</a>

            css
            .teste:active {
                color: orange;
            }

            RESULTADO

            TESTE

Há muitas outras pseudoclasses, como ':empty' quando o elemento não tem filhos, ':first-child' primeiro elemento filho de um elemento, para conhecer mais deles, veja clicando aqui.

SELETORES DE PSEUDOELEMENTOS

Usado para estilizar partes específicas de elementos, como a primeira letra de um elemento, inserir outro elemento ou algo após ou antes de um elemento

Usando o '::first-letter', modificamos a primeira letra de um elemento, algumas mudanças possíveis como propriedade de fonte, cor, margem, borda:


            html
            
            <p class='prg'>Primeira letra do parágrafo</p>

            css

            .prg::first-letter {
                font-size: 3em;
                color: blue;
            }

            RESULTADO

            
   Primeira letra do parágrafo


            Este pseudoelemento só pode ser usado em elemento em nível de bloco.

Usando o '::first-line', modificamos a primeira linha do elemento, algumas mudanças possíveis como propriedade de fonte, cor, margem, borda:

            
            html

            <p class='prg2'></p>

            css

            .prg2::first-line {
                background-color: red;
            }

            RESULTADO:
            
         Lorem ipsum dolor sit, amet consectetur adipisicing elit.
            Optio minus suscipit quae excepturi omnis eius autem sit voluptatibus harum? Inve
            ntore nulla mollitia asperiores error ducimus ut deserunt dicta modi tempore!


            Este pseudoelemento só pode ser usado em elemento em nível de bloco.

Usando o '::after', inserimos algo após o conteúdo especificado, usando a propriedade 'content':


            html

            <p class='contentAfter'>Olá</p>

            css
            
            .contentAfter::after {
                content: '<p>Como vai?</p>'
                color: darkorange;
            }

            RESULTADO

            Olá

Usando o '::before', inserimos algo antes do conteúdo especificado, usando a propriedade 'content':


            html

            <p class='contentBefore'>Olá</p>

            css
            
            .contentAfter::before {
                content: '<p>Hey! </p>'
                color: darkorange;
            }

            RESULTADO

            Olá!

Usando o '::marker', será selecionado os marcadores de listas:


            html

            <ul>
                <li>Um</li>
                <li>Dois</li>
                <li>Três</li>
            </ul>

            css

            ::marker {
                color: blue;
                font-size: 1.5em;
            }

            RESULTADO
            
  • Um
  • Dois
  • Três

Usando o '::selection', permite aplicar algumas mudanças no estado do elemento selecionado, como a cor, plano de fundo, cursor e contorno:


            html

            Lorem ipsum, dolor sit amet consectetur adipisicing elit.
            Reprehenderit iure quisquam placeat, perspiciatis quaerat repudiandae,
            sed molestiae, quas ipsa labore cumque. Similique eveniet quas quam,
            nobis ullam suscipit. Dolorem, aperiam!
            
            css

            .select::selection {
                color: green;
            }

            RESULTADO (Selecione palavras)

            Lorem ipsum, dolor sit amet consectetur adipisicing elit.
            Reprehenderit iure quisquam placeat, perspiciatis quaerat repudiandae,
            sed molestiae, quas ipsa labore cumque. Similique eveniet quas quam,
            nobis ullam suscipit. Dolorem, aperiam!

SELETORES DE ATRIBUTO

Permite estilizar elementos com atributos específicos, usando "[]":

Podemos selecionar todos os elementos com um atributo específico:


            html

            <a class='all'>Hyperlink</a>
            
            <a class='all'>Hyperlink</a>

            <a class='all'>Hyperlink</a>
            
            css

            a[class='all'] {
                color: red;
            }

Podemos selecionar elementos com valores específicos:


            html

            <a class='one'>Hyperlink</a>
            
            <a class='zero'>Hyperlink</a>

            <a class='one'>Hyperlink</a>
            
            css

            a[class='one'] {
                color: orange;
