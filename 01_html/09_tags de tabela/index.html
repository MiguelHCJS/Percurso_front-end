TAG TABLE, CAPTION, TR, TH E TD

  • A tag 'table' define a área da tabela, ela que engloba as demais relacionadas a tabela.
  • A tag 'caption' define a legenda(título) ou assunto da table.
  • A tag 'tr' define a linha em uma tabela, a linha horizontal.
  • A tag 'th' define o cabeçalho da coluna dos dados subsequentes.
  • A tag 'td' define uma celula.
    <table>
        <caption>Dados de cadastro</caption>
        <tr>
            <th>Nome</th>
            <th>Sobrenome</th>
            <th>Idade</th>
        </tr>
        <tr>
            <td>Miguel</td>
            <td>Tenorio</td>
            <td>25</td>
        </tr>
    </table>

TAG THEAD, TBODY E TFOOT

  • A tag 'thead' define a área de cabeçalho em uma tabela.
  • A tag 'tbody' define a área de corpo da table.
  • A tag 'tfoot' define o rodapé da tabela.

    <table>
        <caption>Dados de cadastro</caption>
        <thead>
            <tr>
                <th>Nome</th>
                <th>Sobrenome</th>
                <th>Idade</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Miguel</td>
                <td>Tenorio</td>
                <td>25</td>
            </tr>
            <tr>
                <td>Carlos</td>
                <td>Santos</td>
                <td>25</td>
            </tr>
        </tbody>
        <tfoot>
            <tr>
                <th>Total</th>
                <th colspan='2'>Idade média</th>
            </tr>
            <tr>
                <td>2</td>
                <td colspan='2'>25</td>
            </tr>
        </tfoot>
    </table>
