TAG TABLE, CAPTION, TR, TH E TD

  • A tag 'table' define a área da tabela, ela que engloba as demais relacionadas a tabela.
  • A tag 'caption' define a legenda(título) ou assunto da table.
  • A tag 'tr' define a linha em uma tabela, a linha horizontal.
  • A tag 'th' define o cabeçalho da coluna dos dados subsequentes.
  • A tag 'td' define uma celula.
    <table>
        <caption>Dados de cadastro</caption>
        <tr>
            <th>Nome</th>
            <th>Sobrenome</th>
            <th>Idade</th>
        </tr>
        <tr>
            <td>Miguel</td>
            <td>Tenorio</td>
            <td>25</td>
        </tr>
    </table>
