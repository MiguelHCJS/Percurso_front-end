TAG HEAD

A tag 'head' é onde fica os metadados(title, style, link...) e o que é posto nesta tag não é exposto em display, fica na tag 'html' e antes do 'body':

    <!DOCTYPE html>
    <html>
    
    <head>
        ÁREA DA TAG HEAD, LOCAL DOS METADATOS.
    </head>
    
    <body>

    </body>

    </html>

TAG META

Tag que indica dados adicionais para as informações no documento, como indicar a tabela de caracteres usada no site, indicar autor e descrição de página:

    <!DOCTYPE html>
    <html>
    
    <head>
        <meta charset='UTF-8>
        <meta name='author' content='Miguel Tenorio'>
        <meta name='description' content='demonstração de tags do HTML5'>
    </head>
    
    <body>

    </body>

    </html>

TAG BASE

Esta tag é usada como base para links na página, deve haver apenas um base na página e colocado na tag 'head':

        <!DOCTYPE html>
    <html>
    
    <head>
        <meta charset='UTF-8>
        <meta name='author' content='Miguel Tenorio'>
        <meta name='description' content='demonstração de tags do HTML5'>
        <base href='https://www.w3schools.com/' target='_blank'>
    </head>
    
    <body>
        <a href='tags/tag_base.asp'>Link para mais detalhes</a>
    </body>

    </html>
