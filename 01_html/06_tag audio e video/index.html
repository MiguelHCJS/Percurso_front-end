TAG DE ÁUDIO

Há algumas recomendações para formatos de media de áudio na W3Schools.
Nesta postagem, usarei media mp3: 

    <audio controls>
        <source src="audios/Toddler Band - Godmode.mp3" type="audio/mp3">
    </audio>

É possível colocar mais de um 'source', dentro da tag com formatos de áudios diferentes.
Também é importante verificar sempre os direitos autorais dos áudio.

Existe outros atribulos que podem ser colocados na tag 'audio', no exemplo acima apenas foi atribuido 'controls', servindo para
como o nome já diz, por botões de controle na media.

TAG SOURCE E TRACK

A tag 'source' como já explícita acima, serve para selecionar várias medias, para um elemento de media <audio> ou <video>:

    <audio controls>
        <source src="dog.ogg" type="audio/ogg">
        <source src="dog.mp3" type="audio/mpeg">
        Áudio não suportado pelo navegador.
    </audio>

A tag 'track' é usada para especificar legendas, arquivos de legenda ou outros arquivos que contenham texto, segundo a W3Schools:

    <video width="" height="" controls>
        <source src="forest.mp4" type="video/mp4">
        <source src="forest.ogg" type="video/ogg">
        <track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English">
        <track src="subtitles_no.vtt" kind="subtitles" srclang="no" label="Norwegian">
    </video>

TAG DE VIDEO

Define um conteúdo de vídeo, que também pode ser usada a tag 'source' para especificar várias medias para o mesmo elemento de vídeo:

    <video width="" height="" controls>
        <source src="movie.mp4" type="video/mp4">
        <source src="movie.ogg" type="video/ogg">
        Vídeo não suportado pelo navegador.
    </video>
