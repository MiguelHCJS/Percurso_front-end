TAG CANVAS

Usada para desenhar gráficos via comandos JavaScript, contendo também outras possibilidades de desenhos.

    <canvas id='
identificador
' width='
largura
' height='
altura
'></canvas>

    Código em JavaScript, referente ao desenho abaixo:

    var identificador = document.querySelector('#identificador');
    var d = identificador.getContext('2d');
    d.fillStyle = '#8a2be2';
    d.fillRect(0,0,100,200);
    d.fillStyle = '#ff0031';
    d.fillRect(100,0,200,200)
    d.fillStyle = '#6d31a5';
    d.fillRect(300,0,200,200);

Esta é um tag bem ampla, com uma gama enorme de possibilidades, link de referencia da W3Schools


TAG FIGURE E FIGCAPTION

A tag 'figure' é usada para definir uma área que pode conter ilustrações, fotos, diagramas e etc.

A tag 'figcaption' define uma legenda para o que foi inserido na tag 'figure'

    <figure>
        <img src='
local da imagem
' alt='
texto alternativo
'>
        <figcaption>
legenda da imagem
</figcaption>
    </figure>

TAG PICTURE

Proporciona mais flexibilidade mediante a dimensão da tela, podendo definir mais de uma imagem em diversas dimensões.

    <picture>
        <source media='
podendo definir uma largura, máxima ou mínima
' srcset='
local da imagem
'>
        <source media='' srcset=''>
        <img src='' alt=''>
    </picture>

A tag 'img' será a padrão em caso de uma dimensão não especificada nas outras.

TAG SVG

São gráficos vetoriais escalonáveis, também desenham formas como o canvas, mas diferente dele, tem uma qualidade melhor mesmo se o dimencionar,
possuindo também uma flexibilidade melhor para animações, mais detalhes em Tutorial SVG na W3Schools.

" stroke-width="

"/>
    </svg>
