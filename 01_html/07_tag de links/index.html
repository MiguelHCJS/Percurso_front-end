TAG A

Esta tag é usada para vincular uma página a outra, é importante especificar o atributo 'href', do contrario será apenas um espaço reservado para o link:

    <a href='https://www.w3schools.com/default.asp'>W3Schools<a>

    - Resultado: W3Schools

    - Resultado com o atributo para abrir em uma nova aba: w3schools

TAG LINK

Usada para vincular página com recursos externos, folhas de estilo, sendo uma tag sem fechamento contendo apenas atributos:

    <a rel='
tipo de relacionamento do link
' href='
link para o recurso externo
'>

    - Exemplo link a folha de estilo: <link rel="stylesheet" href="style.css">

TAG NAV

Especifica um conjunto de vinculos de navegação, em outras palavras, define um menu. Sua personalização é mais rica com CSS:

    <nav>
        <a href='https://www.w3schools.com/html/default.asp'>HTML</a>
        <a href='https://www.w3schools.com/css/default.asp'>CSS</a>
        <a href='https://www.w3schools.com/js/default.asp'>JavaScript</a>
    </nav>
Resultado sem CSS:
HTML CSS JavaScript
Resultado com CSS:
HTML CSS JavaScript
