ESTILIZANDO TAGS EM HTML

A linguagem de estilização, mais conhecida como CSS(Cascading style sheet) é usada para estilizar as tags em HTML, sendo possível conectar o arquivo de estilização de três formas:

  • Diretamente na tag
  • No 'head' do arquivo index, usando a tag 'style'
  • Externo em um arquivo CSS
    1. A forma direta na tag, é como usei acima:

        <ul style="list-style: decimal;">
            <li>Diretamente na tag</li>
            <li>No 'head' do arquivo index, usando a tag 'style'</li>
            <li>Externo em um arquivo CSS</li>
        </ul>

    2. Usando a tag 'style' na tag 'head' do index, onde fica as "configurações" do arquivo:

        <head>
            <meta charset='UTF-8'>
            <title>Link interno de estilo</title>
                <style>
                    p { color: red;
                    }
                </style>
        </head>
    
    3. Link externo, também sendo necessário uma tag no 'head', para indicar qual arquivo será usado:

        <head>
            <meta charset='UTF-8'>
            <title>Link interno de estilo</title>
            <link rel='stylesheet' href='definir local e arquivo de estilização'>
        </head>
