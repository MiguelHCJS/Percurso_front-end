ESTILIZANDO TAGS EM HTML

A linguagem de estilização, mais conhecida como CSS(Cascading style sheet) é usada para estilizar as tags em HTML, sendo possível conectar o arquivo de estilização de três formas:

  • Diretamente na tag
  • No 'head' do arquivo index, usando a tag 'style'
  • Externo em um arquivo CSS
    1. A forma direta na tag, é como usei acima:

        <ul style="list-style: decimal;">
            <li>Diretamente na tag</li>
            <li>No 'head' do arquivo index, usando a tag 'style'</li>
            <li>Externo em um arquivo CSS</li>
        </ul>

    2. Usando a tag 'style' na tag 'head' do index, onde fica as "configurações" do arquivo:

        <head>
            <meta charset='UTF-8'>
            <title>Link interno de estilo</title>
                <style>
                    p { color: red;
                    }
                </style>
        </head>
    
    3. Link externo, também sendo necessário uma tag no 'head', para indicar qual arquivo será usado:

        <head>
            <meta charset='UTF-8'>
            <title>Link interno de estilo</title>
            <link rel='stylesheet' href='definir local e arquivo de estilização'>
        </head>

TAG DIV E SPAN

Existe dois tipos de tag, a 'inline element' e a 'block-level', a tag 'div' é de nível de bloco(block) a 'span' é elemento 'em linha'(inline element):

Tag div, com uma altura e largura definida em 200px

<div style="height: 200px; width: 200px; background-color: brown;"></>

Tag span, em um paragrafo:

<span class="destaque">consectetur</span>

TAG HEADER, MAIN E FOOTER

As tags 'header', 'main' e 'footer' vieram para dar um valor mais semantico para a estrutura do arquivo HTML:

  • header define o cabeçalho da página
  • main define o conteúdo principal da página
  • footer define o rodapé da página
    <header>
        <h1>TÍTULO</h1>
    </header>
    <main>
        conteúdo da página
    </main>
    <footer>
        Rodapé comumente usado para por contatos e etc... 
    </footer>

TAGS SECTION, ARTICLE E ASIDE

  • A tag 'section' define uma seção no HTML, de forma que é mais semantico que usar um 'div'.
  • A tag 'article' define uma parte no HTML que de certa forma é independente do assunto em questão.
  • A tag 'aside' define um conteúdo lateral, geralmente como um menu lateral.

TAG DETAILS E SUMMARY

A tag 'details' é utilizada para adicionar detalhes adicionais sobre algo, no qual este detalhe pode ser, ocultado e exibido sob demanda do usuário.
A tag 'summary' para definir um cabeçalho para 'details'. 


    <details>
        <summary>PRATICA DE TAGS</summary>
        <p>Ao completar as informações sobre as tags HTML segundo a listagem da W3Schools, partirei para a prática junto ao CSS.</p>
    </details>

TAG DATA

Esta tag especifica um caminho base para os links associados as tags na página, logo deve haver apenas uma tag 'base' no documento.
Usada para adicionar um valor legível por maquina, junto a um valor legível por humano, renderizado no navegador. 

    <ul>
        <li><data value="1">Dado 1 com valor 1</data></li>
        <li><data value="2">Dado 2 com valor 2</data></li>
        <li><data value="3">Dado 3 com valor 3</data></li>
    </ul>
  • Dado 1 com valor 1
  • Dado 2 com valor 2
  • Dado 3 com valor 3
