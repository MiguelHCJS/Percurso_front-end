TAGS UL, OL E LI

- A tag 'ul' é uma lista não ordenada
- A tag 'ol' é uma lista ordenada
- A tag 'li' é os itens que compõe tanto a lista 'ul' quanto a 'ol' 

    TAG UL 

    <ul>
        <li>Carro</li>
        <li>Avião</li>
        <li>Navio</li>
    </ul> 

    TAG OL 

    <ol>
        <li>Terra</li>
        <li>Ar</li>
        <li>Água</li>
    </ol>
Seguindo modelo acima:

Lista não ordenada

  • Carro
  • Avião
  • Navio

Lista ordenada

  1. Terra
  2. Ar
  3. Água

TAGS DL, DT E DD

- A tag 'dl' define uma lista de definição, em outras palavras, uma lista que indica a definição dos termos nela contida
- A tag 'dt' define o termo, palavra ou algo a ser definido em uma 'dl'
- A tag 'dd' serve para definir o que foi colocado em 'dt' 

    <dl>
        <dt>Carro</dt>
        <dd>Veículo de quatro rodas e diversos modelos e marcas.</dd>
        <dt>Avião</dt>
        <dd>Transporte aéreo com asas e de grande velocidade.</dd>
        <dt>Navio</dt>
        <dd>Transporte turístico ou para uso comercial.</dd>
    </dl>
