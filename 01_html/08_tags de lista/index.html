TAGS UL, OL E LI

- A tag 'ul' é uma lista não ordenada
- A tag 'ol' é uma lista ordenada
- A tag 'li' é os itens que compõe tanto a lista 'ul' quanto a 'ol' 

    TAG UL 

    <ul>
        <li>Carro</li>
        <li>Avião</li>
        <li>Navio</li>
    </ul> 

    TAG OL 

    <ol>
        <li>Terra</li>
        <li>Ar</li>
        <li>Água</li>
    </ol>
Seguindo modelo acima:

Lista não ordenada

  • Carro
  • Avião
  • Navio

Lista ordenada

  1. Terra
  2. Ar
