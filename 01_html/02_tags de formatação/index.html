TEXTO BIDIRECIONAL: <bdi></bdi>
Recebe um texto ou palavra que tenha uma direção diferente do contexto atual
Ex:
Usuário Carlos: 25 anos
Usuário Miguel: 25 anos
Usuário إيان: 30 anos

SUBSTITUIÇÃO BIDIRECIONAL: <bdo></bdo>
Modifica a direção de leitura de uma palavra ou frase:
Ex. sentido esquerda para direita: Miguel Tenorio
Ex. sentido direita para esquerda: Miguel Tenorio

CITAÇÃO CURTA E LONGA:
Citação curta com apenas uma linha, usar <q></q>
Ex: Browsers normally insert quotation marks around the quotation.
Citação longa, usar <blockquote></blockquote>
Ex:

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Vero earum quasi porro deleniti fuga rem hic commodi nostrum alias id soluta quaerat laudantium qui dolore, accusamus reprehenderit corrupti voluptatibus voluptates?

DESTAQUE EM TÍTULOS DE TRABALHOS ARTISTICOS/CRIATIVOS <cite></cite>:
Ex: Mona Lisa de Leonardo Da Vinci de 1503.
Em volta do título da obra, o nome do artista não é título.

Ex: Criando variável em JavaScript: var nome = Miguel.
A palavra var, foi marcada com a tag que refere-se a um código que no caso é do JavaScript.
